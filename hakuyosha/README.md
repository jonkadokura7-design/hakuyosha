# Hakuyosha Website Clone

このプロジェクトは白洋舍のウェブサイトのクローンです。

## 閲覧方法

GitHub Pagesで正しく表示するには、このフォルダ内のすべてのファイル（`index.html`, `images/` フォルダなど）をリポジトリのルート（一番上の階層）に配置してください。

もし `hakuyosha` というフォルダの中に入れている場合は、URLの最後に `/hakuyosha/` を追加してアクセスしてください。
例: `https://[ユーザー名].github.io/[リポジトリ名]/hakuyosha/`
